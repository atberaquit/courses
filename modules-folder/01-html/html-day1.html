Introduction to HTML

Heads-up! Do code-along and lecture to demonstrate the topics below.

Objectives:

  • Understand the purpose and importance of HTML in web development.
  • Learn about the basic structure of an HTML document.
  • Explore text formatting options in HTML.
  • Familiarize with creating lists and linking webpages.
  • Understand the concept of HTML forms and how to create form elements.

Lesson Proper:

Introduction to HTML
  • Definition of HTML and its role in web development.
  • Importance of HTML in structuring web content.
  • Relationship between HTML, CSS, and JavaScript.
HTML Document Structure
  • Overview of the basic structure of an HTML document.
  • Explanation of the doctype declaration, <html>, <head>, and <body> tags.
  • Usage of the <title> tag to specify the webpage title.
Text Formatting
  • Introduction to various HTML tags for text formatting.
  • Usage of heading tags (<h1> to <h6>) for defining different levels of headings.
  • Creating paragraphs using the <p> tag.
  • Inserting line breaks with the <br> tag.
  • Adding horizontal rules using the <hr> tag.
  • Styling text with tags such as <strong>, <em>, <u>, <s>, <sub>, and <sup>.
Lists
  • Overview of different types of lists in HTML.
  • Creating ordered lists using the <ol> and <li> tags.
  • Creating unordered lists using the <ul> and <li> tags.
  • Creating definition lists using the <dl>, <dt>, and <dd> tags.
Links and Anchors
  • Introduction to hyperlinking in HTML using the <a> tag.
  • Usage of the href attribute to specify the destination of the link.
  • Opening links in a new tab or window using the target attribute.
  • Creating anchor points within a page using the <a> tag.
HTML Forms
  • Introduction to HTML forms and their purpose in web development.
  • Explanation of the <form> tag and its attributes.
  • Usage of form elements such as <input>, <textarea>, <select>, and <button>.
  • Understanding the concept of form controls, form validation, and user input.
  • Discussion on advanced form input types: text, number, email, color, date, etc.
  • Usage of attributes like required, pattern, min, max, step, etc.
  • Exploring form validation and error messaging techniques.
  • Introduction to the <select> element for dropdown menus.
  • Creating radio buttons and checkboxes using the <input> element.

Activity:

  1. Create an HTML file named "contact.html".
  2. Copy and paste the provided sample code into the HTML file.
  3. Add the necessary form elements within the <form> tags to create the contact form.
  4. Use appropriate input types such as text, email, number, color, etc., for the form fields.
  5. Include labels for each form element using the <label> tag.
  6. Customize the form by adding any additional form elements or attributes you want.
  7. Experiment with different form attributes like required, pattern, placeholder, etc.
  8. Test the form by filling in the fields and submitting it. Observe the behavior of the form when fields are left empty or when validation patterns are not met.

Sample Code:

<!DOCTYPE>
<html>
<head>
  <title>Contact Form</title>
</head>
<body>
  <h1>Contact Form</h1>

  <form>
    <div>
      <label for="name">Name:</label>
      <input type="text" id="name" name="name" required>
    </div>

    <div>
      <label for="email">Email:</label>
      <input type="email" id="email" name="email" required>
    </div>

    <div>
      <label for="message">Message:</label>
      <textarea id="message" name="message" required></textarea>
    </div>

    <div>
      <label for="subject">Subject:</label>
      <input type="text" id="subject" name="subject">
    </div>

    <div>
      <label for="phone">Phone:</label>
      <input type="tel" id="phone" name="phone" pattern="[0-9]{3}-[0-9]{3}-[0-9]{4}" placeholder="XXX-XXX-XXXX">
    </div>

    <div>
      <label for="priority">Priority:</label>
      <select id="priority" name="priority">
        <option value="low">Low</option>
        <option value="medium">Medium</option>
        <option value="high">High</option>
      </select>
    </div>

    <div>
      <label>Newsletter Subscription:</label>
      <input type="checkbox" id="newsletter" name="newsletter">
    </div>

    <div>
      <label>Gender:</label>
      <input type="radio" id="male" name="gender" value="male">
      <label for="male">Male</label>
      <input type="radio" id="female" name="gender" value="female">
      <label for="female">Female</label>
    </div>

    <button type="submit">Submit</button>
  </form>
</body>
</html>
