HTML 2

Heads-up! Do code-along and lecture to demonstrate the topics below.

Objectives:

  • Understand the concept of semantic HTML and its importance in web development.
  • Learn about advanced form elements and attributes.
  • Explore tables and their usage in structuring data.
  • Gain knowledge about embedding external content using iframes.

Lesson Proper:

Semantic HTML
  • Definition and purpose of semantic HTML
  • Importance of using semantic tags for better accessibility and search engine optimization
  • Examples of semantic tags such as <header>, <nav>, <main>, <section>, <article>, <aside>, <footer>, etc.
  • Guidelines for using semantic HTML effectively
Tables
  • Introduction to HTML tables and their usage in organizing tabular data
  • Creating table structures using the <table>, <tr>, <th>, and <td> tags
  • Merging cells horizontally using the colspan attribute
  • Merging cells vertically using the rowspan attribute
  • Applying table headers using the <thead>, <tbody>, and <tfoot> tags
  • Styling tables using CSS and adding captions with the <caption> tag

Activity:

  1. Create an HTML file named "advanced.html".
  2. Copy and paste the provided sample code into the HTML file.
  3. Design a semantic structure for the webpage using appropriate semantic tags, such as <header>, <main>, <form>, <table>, and <footer>.
  4. Implement an advanced contact form that includes input fields for name, email, and message using the <input> and <textarea> elements. Add the necessary labels using the <label> tag and apply the required attribute for validation.
  5. Create a table to display student grades using the <table>, <thead>, <tbody>, <th>, and <td> tags. Use the colspan and rowspan attributes to merge cells and structure the table accordingly.
  6. Customize the styling of the webpage using CSS to enhance the visual appeal.
  7. Test the webpage by loading it in a web browser and ensuring that all elements are displayed correctly.

Sample Solution:


<!DOCTYPE html>
<html>
<head>
<title>Advanced HTML Page</title>
<style>
    /* CSS styling goes here */
</style>
</head>
<body>
<div class="container">
    <header>
    <h1>Contact Form</h1>
    </header>

    <main>
    <form>
        <label for="name">Name:</label>
        <input type="text" id="name" name="name" required>

        <label for="email">Email:</label>
        <input type="email" id="email" name="email" required>

        <label for="message">Message:</label>
        <textarea id="message" name="message" required></textarea>

        <input type="submit" value="Submit">
    </form>

    <table>
        <caption>Student Grades</caption>
        <thead>
        <tr>
            <th>Student Name</th>
            <th>Subject</th>
            <th>Grade</th>
        </tr>
        </thead>
        <tbody>
        <tr>
            <td rowspan="2">John Doe</td>
            <td>Math</td>
            <td>A</td>
        </tr>
        <tr>
            <td>Science</td>
            <td>B+</td>
        </tr>
        <tr>
            <td>Jane Smith</td>
            <td colspan="2">No Data Available</td>
        </tr>
        </tbody>
    </table>
    </main>

    <footer>
    <p>© 2023 My Website. All rights reserved.</p>
    </footer>
</div>
</body>
</html>
